### T03: Criar funcao auxiliar sanitizeHTML

<read_first>
- `conversas.php` ou arquivo JS compartilhado — ver se ja existe funcao de escape
</read_first>

<action>
Criar funcao `sanitizeHTML(str, allowedTags)` no escopo do arquivo ou em um helper compartilhado:

```javascript
function sanitizeHTML(str, allowedTags = ['b', 'i', 'strong', 'em', 'br', 'p']) {
    // 1. Remover todas as tags
    let cleaned = str.replace(/<[^>]*>/g, (match) => {
        const tagName = match.match(/<\/?(\w+)/);
        if (tagName && allowedTags.includes(tagName[1].toLowerCase())) {
            // Permitir apenas a tag basica (remover atributos)
            const isClosing = match.startsWith('</');
            return isClosing ? `</${tagName[1]}>` : `<${tagName[1]}>`;
        }
        return ''; // Remover tags nao permitidas
    });
    
    // 2. Escapar entidades HTML restantes (previne bypass com &lt;script&gt;)
    const div = document.createElement('div');
    div.textContent = cleaned;
    cleaned = div.innerHTML;  // Converte < para &lt; etc, mas preserva tags validas
    
    // 3. Re-renderizar apenas tags permitidas
    allowedTags.forEach(tag => {
        cleaned = cleaned.replace(
            new RegExp(`&lt;(/?(${tag}))&gt;`, 'gi'),
            '<$1>'
        );
    });
    
    return cleaned;
}
```

Se houver um arquivo JS compartilhado (ex: `assets/auvvo-utils.js`), colocar a funcao la e importar nos arquivos que precisam.
</action>

<acceptance_criteria>
- Funcao `sanitizeHTML` definida e acessivel
- `sanitizeHTML('<script>alert(1)</script>')` retorna string sem tags script
- `sanitizeHTML('<b>texto</b>')` retorna `<b>texto</b>` (preserva tags permitidas)
- `sanitizeHTML('<img src=x onerror=alert(1)>')` retorna string sem `<img>`
- `sanitizeHTML('teste & <b>negrito</b>')` preserva `&` como texto e `<b>` como tag
</acceptance_criteria>
